美麗暨大（首頁／範例1）
引人遐思的美麗「水沙連」，從清初以來就在平埔族人與安雅支族人的口中流傳著。早期，它泛指內山先住民棲息地(tsualihen或soalian)的意思，又合著波光粼粼的日月潭湖水，因此而有「水‧沙連」的美稱。水沙連由連聳的山脈、起伏的丘陵、平坦的盆地和蜿蛻的沒流所構成，垂直高度從海拔三百公尺到三千六百零五公尺之間，涵蓋了寒、溫、熱三帶完整的動植物群落，而量豐沛加上氣候適中，多采多姿的自然環境生態一直吸引各個族群遷來棲息定居，逐漸孕育此地多元文化的特色。
現今的埔里，物產豐饒，加上四季如春的氣候，贏得了「花之都」的雅號。埔里的社區活動十分活絡，讀書會、民俗廟會、丈史藝術工作者的聚會頻繁等，埔里居民有著「社區總體營造」的意識，共同建構一個擁有 濃厚人文色彩的城鎮。
民國八十四年國立暨南國際大學正式成立，座落於埔里通往日月潭的藝術大道旁，從入口蜿蜓而上，一片翠綠盡收眼簾，數棟中國風的建築融合於大自然中，融入水沙連的山水間。暨大擁有豐富的自然景觀，在這裡，晴空萬里時可以遠眺合歡山，朗朗無雲之夜可以觀星，甚至雨後可見清晰的彩虹；霧社、奧萬大、日月潭、新中橫、合歡山、惠蓀林場都在附近；校園內設置人文藝廊，也不時可看見學子在校園的一隅看著書。如此俯拾皆是的校園美景，可以培養學生更廣闊的胸襟、時刻關懷周遭的心思，作為開創未來的做重要資產。利用埔里好山好水的天然資源，規劃出在日月潭遊湖的風帆課、附近馬場的馬術課、埔里素有盛名的射箭課，以及高爾夫、壘球、游泳和國標舞等特色運動課程。
本校以建設多樣化的生態公園環境，作為校園景觀目標，主要願景內容包括：台灣杉生態公園、暨大黑森林、春櫻冬梅杏花林、茄苳樹綠色隧道、孟宗竹林、藝術路標、天際線導覽地標、暨大天籟園、台灣原生植物園步道、中水道生態景觀工程。
校 名 本校校名『暨南』二字取自書經禹貢篇『朔南暨，聲教訖於四海』的含義，其意是要把中華文化向南方的海外傳播。早在一九零六年清季末年設在南京的暨南學堂，及民國以後設在上海的暨南大學，以培育僑界人才為其傳統及特色。本校承襲『暨南』之名，即是希望延續此一使命，以培養海內外優秀人才自期。再者，針對國家社會未來發展的需求，培養具國際觀的高級人才，亦為本校成立最重要的目標。加上『國際』二字，意在與廣州之『暨南大學』有所區隔．準此，本校乃定名為『國立暨南國際大學』。
校 徽 本校校徽以校名暨南 CHI NAN 之「C、N」字首，融合中國傳統 紋飾及地球形設計而成，設計雖結合中國化、現代化， 造成簡潔易明，蘊涵本校設校宗旨、教育理念;將本校得 天獨厚的環境，以更積極創新的作法，為國人開創更美 好的教育遠景。 造形意義 對稱循環的紋飾結構，代表僑界和國內，象徵僑教功能及海外支持力量的雄厚、自強不息。將「C、N」設計襯 托出「中」字造型，躍然地球之上，代表本校富中華文 化「中道」之精神。 外形以「圓」形彰顯出「允執厥中 ，止於至善」之含意。 色彩表現 以穩健、自然、務實的深藍色，為校徽注入更多的生命 和深遠的意義，象徵本校在「誠樸弘毅、務本致用」校 訓的推動之下，必能促使校務，朝向更完善、校和諧、 更美好的境界。
校 園 建 設 規 劃 本校校園廣達 150 公頃 ( 約 1,482,141 平方公尺 ) ， 校園建築及其他硬體設施之規劃，除以美觀實用為主外，並以蘊涵中華文化精神之風格為其特色。在校園總體配置方面，依據教學、研究、服務、居住與休憩等活動之機能而規劃；建築工程則依校務發展需求，分期實施。有關自然環境保育、開放空間、植栽、道路系統及資訊網路管線等，均已分別訂定設計準則，預先規劃。校園建築之基本配置，以連接行政大樓及圖書資訊大樓之徒步廣場與步道為主軸，並以貫穿科技學院之徒步步道為副軸，而在環校道路之配合穿引下，興建主要建築物。
本校校舍建築，以院落建築為基本格局，各建築磚牆色彩力求調和，部分建築並搭配中式斜屋頂，以與校地四周宏偉秀麗之山景呼應；本校在興建校舍之同時，亦重視動線系統與現代化公共設施之配合。例如：綠地、植栽及休憩空間之規劃、污水及垃圾處理設施之建造，以及水電、通訊、資訊網路管線的預留等。本校創校迄至民國 92 學年度，已完成並使用的建築物，包括：行政大樓、綜合教學大樓、科技學院、學生餐廳、學生宿舍 (A 、 B 、 C 、 D 棟 ) 、學人會館、學人宿舍 ( 雙併 14 棟、 34 戶 ) 、職務宿舍 ( 校長、副校長 ) 、臨時學生社團活動中心，以及環校道路、聯外道路、供水設施、供電設施、瓦斯設施、田徑場、資源回收場 ( 原垃圾處理場 ) 和廢水處理場等公共設施。
93年人文學院大樓啟用，95年圖資大樓啟用，管理學院大樓96年完工啟用，學生活動中心、體育健康中心及研究生宿舍(男女兩棟，共426間雙人套房)等重大新建工程亦相繼於98、99年間完工啟用，提供同學更佳的學習、活動與居住空間，101年8月完成教職員宿舍(四層雙併建築物，三房兩廳格局，32戶)，提供並配合校園景觀與休閒工程，漸次朝向建構生態公園和健康校園的方向發展。
暨大經過十多年的建校過程，當初一片牧草地的景象，已由矗立的建物及綠蔭、樹木、花草所取代，現階段校園內可見白鷺鷥飛抵，以及數不盡數量與種類的野鳥來訪，惟原始生物棲息卻因興建而愈來愈少。為落實生態永續的價值思維，未來將透過學校政策、教學、生活及校園空間改造，於校務發展的各環節持續的紮根與實踐。在硬體方面落實「生態環境恢復與維護」、「永續建築」、「省能節源」、「健康建築」、「基地永續」、「減廢」、「資源再利用」，在軟體面落實「校園環境政策」、「建立校園空間建築與環境管理系統」、「推動校園環境教學」、「實施生活環保」，更要強化社區的互動與參與，以本校校園公共空間作為主軸發展，除了凝聚師生的認同與共識，更要建立與南投縣政府的夥伴關係，並利用本地居民參與方式獲致鄰里社區認同，結合具有凸顯地域特色、順應環境條件、凝聚社區意識等的合作效益，同時運用學校的特色，發展出屬於暨大的永續校園環境。
